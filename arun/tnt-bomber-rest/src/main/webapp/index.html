NOTE: X, Y, and Z spreads are in increments of 0.1

REST URIs

  • webresources/tnt/count: to get and set the number of TNTs
  • webresources/tnt/spreadx: to get and set the spread of TNTs along X-Axis
  • webresources/tnt/spready: to get and set the spread of TNTs along Y-Axis
  • webresources/tnt/spreadz: to get and set the spread of TNTs along Z-Axis
